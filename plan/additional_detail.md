## Scope
1. What to do
- Redundancy file protection
- Elastic compatability
- AI MCP
- User interface
2. What no to do
- Multiple Computer distributed system

## Roles
1. Storage Users (human) (there should be like only one user)
- Responsibility
    - Handle their own files
- Permission
    - CRUD or search their own files or files shared with them
2. Agentic AI
- Responsibility
    - Handle their own files
- Permission
    - The same as human users
    - Access to MCP
3. Admin / Hoster (The access sounds weird but this is for personal use so Admin and user might even be the same person)
- Responsibility
    - Handle hosting
    - Manage everone's files
- Permission
    - Everyone's file

## Additional Goals
- The system should be able to access accross the internet.
